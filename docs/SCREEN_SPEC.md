# CampusLog Screen Spec

## Purpose

이 문서는 CampusLog MVP의 핵심 화면별 목적, UI 구성 요소, 입력값, 출력값, 버튼 동작, 빈 상태, 오류 상태를 정리합니다.

## 1. 활동 경험 목록 화면

### 목적

사용자가 저장한 활동 경험을 한눈에 확인하고, 새 경험을 기록하거나 기존 경험으로 이동할 수 있게 합니다.

### UI 구성 요소

- 화면 제목
- 활동 경험 카드 목록
- 새 경험 추가 버튼
- 각 경험의 제목, 기간, 역할, 주요 키워드, AI 분석 상태

### 입력값

- 없음

### 출력값

- 저장된 활동 경험 목록

### 버튼 동작

- 새 경험 추가: 활동 경험 작성 화면으로 이동
- 경험 카드 선택: 활동 경험 상세 / 수정 화면으로 이동
- AI 분석 보기: 해당 경험의 AI 분석 결과 화면으로 이동

### 빈 상태

- 저장된 경험이 없으면 "아직 기록한 활동 경험이 없습니다" 메시지와 새 경험 추가 버튼을 보여줍니다.

### 오류 상태

- 저장된 데이터를 불러오지 못하면 다시 시도 안내를 보여줍니다.

## 2. 활동 경험 작성 / 수정 화면

### 목적

사용자가 프로젝트, 공모전, 인턴, 대외활동 등 자신의 활동 경험을 직접 입력하거나 수정할 수 있게 합니다.

### UI 구성 요소

- 제목 입력
- 기간 입력
- 역할 입력
- 내용 입력
- 성과 입력
- 관련 링크 입력
- 저장 버튼
- 취소 버튼
- AI 분석 요청 버튼

### 입력값

- 제목
- 기간
- 역할
- 내용
- 성과
- 관련 링크

### 출력값

- 저장된 활동 경험

### 버튼 동작

- 저장: 입력한 경험을 저장하고 목록 또는 상세 화면으로 이동
- 취소: 입력 내용을 저장하지 않고 이전 화면으로 이동
- AI 분석 요청: 저장된 경험을 기준으로 AI 경험 분석 결과 화면으로 이동

### 빈 상태

- 새 경험 작성 시 입력 필드는 비어 있는 상태로 보여줍니다.

### 오류 상태

- 필수 입력값이 비어 있으면 저장하지 않고 안내 메시지를 보여줍니다.
- 관련 링크 형식이 올바르지 않으면 링크 확인 안내를 보여줍니다.

## 3. AI 경험 분석 결과 화면

### 목적

AI가 활동 경험을 요약하고, 핵심 역량, 주요 성과, 활용 가능한 키워드를 보여줍니다.

### UI 구성 요소

- 경험 요약 영역
- 핵심 역량 태그
- 주요 성과 목록
- 활용 가능한 키워드
- 다시 분석 버튼
- 활용 지원으로 이동 버튼

### 입력값

- 저장된 활동 경험

### 출력값

- 경험 요약
- 핵심 역량
- 주요 성과
- 활용 가능한 키워드

### 버튼 동작

- 다시 분석: 같은 경험을 기준으로 AI 분석을 다시 요청
- 활용 지원으로 이동: AI 경험 추천 및 활용 화면으로 이동

### 빈 상태

- 아직 분석 결과가 없으면 AI 분석 요청 버튼을 보여줍니다.

### 오류 상태

- AI 분석에 실패하면 실패 안내와 다시 시도 버튼을 보여줍니다.

## 4. AI 경험 추천 및 활용 화면

### 목적

사용자가 입력한 자기소개서 문항, 포트폴리오 작성 목적, 면접 질문에 맞춰 가장 적합한 경험 1개를 추천하고, 추천 이유와 참고 문장을 제공합니다.

### UI 구성 요소

- 활용 목적 선택
- 질문 / 문항 입력창
- 추천 요청 버튼
- 추천 경험 카드 1개
- 추천 이유
- 강조할 역량과 성과
- 자기소개서 / 포트폴리오 / 면접 참고 문장
- 복사 버튼
- 저장 버튼

### 입력값

- 자기소개서 문항
- 포트폴리오 작성 목적
- 면접 질문
- 저장된 활동 경험과 AI 분석 결과

### 출력값

- 가장 적합한 활동 경험 1개
- 해당 경험이 적합한 이유
- 강조하면 좋은 역량과 성과
- 활용 가능한 참고 문장

### 버튼 동작

- 추천 요청: 입력한 목적에 맞는 경험 추천과 참고 문장 생성을 요청
- 복사: 참고 문장을 클립보드에 복사
- 저장: 추천 결과를 브라우저 LocalStorage에 저장

추천 결과 저장은 MVP에 포함하지만, 첫 구현에서는 복사 기능보다 낮은 2순위로 다룹니다. 여러 추천 후보를 탐색하는 기능은 첫 프로토타입에서 제외합니다.

### 빈 상태

- 저장된 활동 경험이 없으면 먼저 활동 경험을 기록하라는 안내를 보여줍니다.
- 질문 / 문항이 비어 있으면 입력 안내를 보여줍니다.

### 오류 상태

- AI 추천에 실패하면 실패 안내와 다시 시도 버튼을 보여줍니다.
- 복사에 실패하면 직접 선택해서 복사할 수 있도록 안내합니다.

## MVP Excluded Screens

이번 MVP에서는 아래 화면을 만들지 않습니다.

- 로그인 / 회원가입 화면
- 결제 화면
- 커뮤니티 화면
- 관리자 화면
- 공개 프로필 화면
- 외부 서비스 자동 연동 설정 화면
- PDF 포트폴리오 자동 생성 화면
- 여러 추천 후보 탐색 화면
