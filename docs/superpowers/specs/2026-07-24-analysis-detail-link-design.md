# AI 분석 스플릿뷰 상세 이동 디자인

## 목표

`나의 활동`의 AI 분석 결과 스플릿뷰에서 독립 분석 상세 화면으로 이동할 수 있는 명확한 진입점을 제공한다.

## 결정

- 분석 스플릿뷰 하단 왼쪽에 `분석 상세 보기` 버튼을 추가한다.
- 버튼은 활동 상세의 `활동 상세 보기`와 같은 진한 primary 프레임과 오른쪽 `ArrowRight` 아이콘을 사용한다.
- 버튼은 `/experiences/[id]/analysis`로 이동한다.
- 기존 `다시 분석하기`는 같은 하단 행의 오른쪽에 유지한다.
- 좁은 화면에서도 두 액션은 서로의 의미를 침범하지 않고 가로 overflow를 만들지 않도록 한다.

## 영향 범위

- `DashboardAnalysisSplitPanel`의 하단 탐색 액션만 변경한다.
- 분석 요청, 재분석, 저장, API와 데이터 계약은 변경하지 않는다.

## 검증

- 구조 테스트로 링크 경로, 문구, 화살표, 버튼 순서를 확인한다.
- 실제 로그인 `/experiences` 스플릿뷰에서 두 버튼 배치와 가로 overflow를 확인한다.
- lint, typecheck, 관련 테스트와 `git diff --check`를 실행한다.
