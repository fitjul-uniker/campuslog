# AI 분석 부족 정보 하단 메타 간소화

## 목표

열린 부족 정보 질문의 하단에서 기본 상태 `답변 없음`과 노출형 키보드 안내 `⌘/Ctrl + Enter`를 제거해 저장 액션 주변의 시각적 잡음을 줄입니다.

## 승인된 구성

- 답변 글자 수 `현재 글자 수/1600`은 항상 표시합니다.
- `저장 중`, `저장 실패`, `기존 답변 수정 중`, `답변 작성 중`, `저장 완료`, 마지막 저장 시각처럼 실제 상태 변화가 있을 때만 상태 문구를 표시합니다.
- 답변이 없고 작성도 시작하지 않은 기본 상태에서는 상태 문구를 표시하지 않습니다.
- `⌘/Ctrl + Enter` 안내 문구는 화면에서 제거합니다.
- Command/Ctrl+Enter 저장 동작 자체는 유지합니다.
- 저장 버튼, 오류 안내, 답변 저장 repository·API·schema 계약은 변경하지 않습니다.

## 성공 기준

- 빈 답변의 열린 질문 하단에는 `0/1600`과 `답변 저장`만 표시됩니다.
- 작성·저장·오류 상태는 기존 상태 문구와 오류 안내를 계속 제공합니다.
- 키보드 단축키 저장 회귀 테스트가 유지됩니다.
