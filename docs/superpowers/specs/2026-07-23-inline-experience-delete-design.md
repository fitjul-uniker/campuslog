# 나의 활동 인라인 상세 삭제 액션 설계

## 범위

`/experiences`에서 완료 경험을 선택했을 때 오른쪽에 열리는 인라인 활동 상세 하단 액션 줄에 삭제 기능을 추가합니다. 진행 활동 상세와 독립 `/experiences/[id]` 상세의 동작은 변경하지 않습니다.

## 화면

액션 순서는 다음과 같습니다.

1. 활동 상세 보기
2. 수정
3. 휴지통 아이콘과 `삭제` 텍스트
4. AI 분석 요청 또는 AI 분석 결과

데스크톱의 좁은 상세 슬롯에서도 네 액션을 한 줄로 유지합니다. 모바일에서는 기존 2열 액션 배치를 따릅니다.

## 동작

- 삭제 선택 시 기존 완료 경험 삭제 확인 문구를 표시합니다.
- 확인하면 repository의 경험 삭제를 실행해 연결 분석·추천 데이터 정리 계약을 그대로 사용합니다.
- 성공하면 해당 경험을 목록에서 제거하고 오른쪽 상세를 닫습니다.
- 실패하면 목록과 상세를 유지하고 오류 안내를 표시합니다.

## 접근성

- 실제 `button` 요소와 `Trash2` 아이콘을 사용합니다.
- 버튼의 보이는 `삭제` 텍스트를 접근성 이름으로 사용합니다.
- 키보드 focus-visible과 기존 액션 최소 높이를 유지합니다.

## 검증

- 인라인 상세에 네 액션이 모두 렌더링되는 구조 회귀 테스트
- 삭제 취소 시 목록과 상세 유지
- 삭제 성공 시 목록 제거와 상세 닫힘
- lint, typecheck, build
- 데스크톱 좁은 상세 슬롯과 모바일에서 가로 잘림 확인
