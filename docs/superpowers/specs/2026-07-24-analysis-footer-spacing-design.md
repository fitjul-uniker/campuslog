# AI 분석 하단 액션 여백 디자인

## 목표

독립 AI 분석 결과 카드와 하단 `다시 분석하기` 버튼이 지나치게 붙어 보이지 않도록 시각적 간격을 정리한다.

## 결정

- 기존 왼쪽 정렬과 버튼 디자인은 유지한다.
- 분석 결과 영역과 하단 액션 사이에 24px 여백을 둔다.
- 구분선이나 별도 배경은 추가하지 않는다.
- 분석 결과가 없는 상태에서는 버튼이 이미 상세 패널 내부에 있으므로 불필요한 이중 여백을 만들지 않는다.

## 영향 범위

- `analysis-page-footer-actions`가 분석 결과 카드 다음에 나오는 경우에만 상단 여백을 적용한다.
- 분석 API, 저장 데이터, 상태 처리와 버튼 동작은 변경하지 않는다.

## 검증

- 구조 테스트에서 하단 액션 전용 상단 여백을 확인한다.
- 로그인된 독립 분석 화면에서 버튼과 결과 카드 간격, 가로 overflow를 확인한다.
- lint, typecheck, 관련 구조 테스트와 `git diff --check`를 실행한다.
